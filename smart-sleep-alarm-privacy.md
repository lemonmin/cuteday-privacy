  # 개인정보처리방침 — Smart Sleep Alarm

  시행일: 2026-05-15

  본 방침은 모바일 앱 "Smart Sleep Alarm"(패키지명: `smart.sleep.alarm`,
  이하 "앱")이 사용자의 개인정보를 어떻게 처리하는지 설명합니다.

  ## 1. 앱이 수집·저장하는 정보

  본 앱은 **사용자가 직접 입력한 알람·타이머·규칙 설정**(시각, 요일,
  사운드 모드, 벨소리 URI, 미션 종류, 패턴 코드 등)을 **사용자 기기 내부**의
  SharedPreferences에 저장합니다. 이 정보는 **외부 서버로 전송되지 않으며**,
  사용자 본인 외 누구도 접근할 수 없습니다. 앱 삭제 시 함께 삭제됩니다.

  본 앱은 **이름·이메일·전화번호·주소 등 사용자를 식별할 수 있는
  개인정보를 수집하거나 서버에 전송하지 않습니다.**

  ## 2. 사용 권한

  | 권한 | 사용 목적 |
  |---|---|
  | 정확한 알람 (USE_EXACT_ALARM, SCHEDULE_EXACT_ALARM) | 사용자가 설정한 시각에 정확히 알람을 울리기 위함 |
  | 알림 (POST_NOTIFICATIONS) | 알람 발화 알림 표시 |
  | 전체화면 인텐트 (USE_FULL_SCREEN_INTENT) | 잠금화면 위 알람 표시 |
  | 부팅 완료 수신 (RECEIVE_BOOT_COMPLETED) | 재부팅 후 알람 재예약 |
  | 진동 (VIBRATE) | 진동 알람 |
  | WakeLock | 알람 발화 시 화면 점등 유지 |
  | 방해금지 접근 (ACCESS_NOTIFICATION_POLICY) | "규칙" 기능에서 알림 무음 전환 (사용자가 별도 허용) |
  | 시스템 설정 수정 (WRITE_SETTINGS) | "규칙" 기능에서 화면 밝기 제한 (사용자가 별도 허용) |
  | 인터넷 / 네트워크 상태 (INTERNET, ACCESS_NETWORK_STATE) | 광고(AdMob) 표시 |
  | 광고 ID (AD_ID) | 광고 맞춤 표시 |

  ## 3. 카메라 사용 (셀카 미션)

  "셀카 찍기" 미션 선택 시, 본 앱은 **Android 시스템 카메라 앱을
  별도 인텐트로 호출**합니다. 본 앱은 카메라 권한을 직접 요청하지
  않으며, 촬영된 사진은 **시스템 카메라 앱이 사용자 갤러리에 저장**합니다.
  본 앱은 사진을 외부로 전송하지 않으며, 미션 통과 여부 판정 후
  앱 메모리에서 제거됩니다.

  ## 4. 제3자 제공 — Google AdMob

  본 앱은 광고 수익 모델을 사용하며, **Google AdMob**을 통해
  배너/전면/보상형 광고를 표시합니다. 광고 SDK는 광고 표시·맞춤 광고·
  사기 방지를 위해 **광고 ID, IP 주소, 기기 정보, 광고 상호작용 데이터** 등을
  수집할 수 있습니다. 자세한 내용은 Google의 정책을 따릅니다:

  - Google 개인정보처리방침: https://policies.google.com/privacy
  - AdMob 데이터 사용:    https://support.google.com/admob/answer/6128543

  사용자는 기기 설정 → "광고" → "광고 ID 재설정" 또는 "맞춤 광고
  선택 해제"를 통해 맞춤 광고를 비활성화할 수 있습니다.

  ## 5. 어린이 (13세 미만) 정책

  본 앱은 13세 미만 어린이를 대상으로 하지 않습니다. 13세 미만의
  개인정보를 의도적으로 수집하지 않습니다.

  ## 6. 보안

  기기 내 저장 데이터는 Android 운영체제가 제공하는 표준 샌드박스
  보호 안에 보관되며, 다른 앱이나 외부에서 접근할 수 없습니다.
